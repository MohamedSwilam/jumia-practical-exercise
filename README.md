## Installation

### 1. Create .env file from .env.example and setup database information

### 2. Install Composer
```bash
composer install
```

### 3. Install npm in root directory and frontend directory
```bash
npm install
```
```bash
cd frontend
```
```bash
npm install
```
### 4. Compile VueJS
```bash
npm run dev
```
